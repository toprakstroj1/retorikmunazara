# RETORİK Form Entry Aracı

Bu klasör ayrı bir Netlify sitesi olarak yüklenebilir.

## Netlify ayarı

Base directory:

```text
form-entry-araci
```

Build command:

```text
npm install
```

Publish directory:

```text
.
```

## Kullanım

1. Google Form linkini yapıştır.
2. `Soruları Çıkar` düğmesine bas.
3. Sağdaki JSON çıktısını kopyala.
4. Telegram botu `/yeni` akışında delege veya organizasyon form bilgisi istediğinde bu JSON'u gönder.

Form herkese açık değilse araç soruları okuyamaz.
